Course Descriptions and Workshops

Seattle University- Web Applications and Technology

Foundations of the Web

This course empowers future developers with the attitude and abilities to shape and control that tool, ensuring that through their service the web continues to serve everyone. It is intended to provide a broad base of understanding for concepts and techniques that will come up over and over during the process of building for the web, working on web production teams, and taking future courses in web development.

Introduction to Web Development

This course introduces students to the basic principles of web Development beginning with an introduction to HTML and CSS. Students will learn how to create static pages and how to implement designs consistent with web Standards and best practices. Students will also learn about tools to give them feedback on the quality of their HTML and CSS in order to insure accessibility and accuracy. Finally, students will practice techniques to insure consistency of presentation across different client browsers.

Introduction to Javascript Programming for the Web

In this course, I learned how to write JavaScript and how to use it on web sites to make engaging, interactive pages. This course will stretch your skills in terms of logical thought, observation, and understanding how the pieces of a web page go together.This course introduces you to the basics of JavaScript and its use in building websites. You will learn about the basic principles of programming, and apply them in JavaScript projects.

Introduction to Servers and Hosting

This course gave me a useful overview of the ways websites are hosted on web servers and delivered to users. I was able to focus on how to use the tools that are commonly available and provided by various hosting companies. From this course, I gained a foundational ability to use command-line Unix, to modify and operate different deployments. I also learned common command-line utilities, worked with command-line text editors, and moved files around without the use of graphical user interfaces.

Building Javascript Applicaitons for the Web

In this course, I studied how modern web applications are built using HTML, CSS and Javascript. I used contemporary development tools including Node.js, Yeoman, Bower, and Grunt to build an AngularJS application that leveraged a third party API. See (NFLDocket App) I also explored the world of local development, file pre-processors, and a number of modern approaches to CSS including SCSS/SASS.

User Science and Testing

In this course I studied at the various ways of how users are interact with my websites and applications, and what I can do to enhance my sites and applications in light of those discoveries. I analyzed how users use my web pages, which was crucial to building effective sites that engage users and fulfill their requirements. Throughout this course, I published my tests, findings and analytics regarding usability and accessibility on my User Science Journal which is a Wordpress based site.
